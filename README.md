# CyberShield Toolkit

A Python-based cybersecurity utility suite developed as part of the Open Source Software Development (OSSD) course.

CyberShield Toolkit combines multiple cybersecurity tools into a single desktop application built with Python and Tkinter. The project demonstrates practical cybersecurity concepts while applying open-source collaboration practices such as GitHub Issues, Branching, Pull Requests, and Code Reviews.

---

## Features

### Password Strength Analyzer

* Password strength evaluation
* Security scoring
* Entropy estimation
* Live analysis
* Password breach simulation
* Security recommendations

### Password Generator

* Secure password generation using Python's `secrets` module
* Custom password length
* Uppercase, lowercase, numbers, and special characters
* Entropy estimation
* Password strength indicators

### File Hash / Integrity Checker

* File integrity verification
* Hash generation
* Hash comparison
* Support for common hashing algorithms

### Phishing Email Detector

* Suspicious keyword detection
* URL analysis
* Sender analysis
* Risk scoring system
* Phishing classification

## Installation

Clone the repository:

```bash
git clone https://github.com/Sikandarr56/CyberShield-Toolkit.git
```

Navigate to the project directory:

```bash
cd CyberShield-Toolkit
```

Run the application:

```bash
python main.py
```
